## 実装した表示構造

- スマートフォン: 会話、作業、差分、ファイル、実行の下部ナビゲーションと1カラム表示。
- タブレット: workspace railと作業表示を2列にする。
- PC: workspace rail、中央作業領域、状態パネルの3領域にする。
- 全端末: 接続状態、Active work、読み取り専用ラベル、開始不可の理由を文字列でも表示する。

2. 768px幅相当で、workspace railと作業表示が読めること。
